# Tools

- read, write, edit: Configuration files (Dockerfile, docker-compose, CI/CD, wrangler)
- exec, bash: Docker builds, CI/CD testing, infrastructure commands, wrangler deploys
- memory_search: Retrieve infrastructure decisions

- Cloudflare Workers / Wrangler for edge deployment
