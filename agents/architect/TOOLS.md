# Tools

## Primary Tools
- read, write, edit: Create and maintain design documents
- web_search, web_fetch: Research technologies, libraries, and best practices
- memory_search: Retrieve past architecture decisions
- sessions_send: Communicate design specs to @Sage for distribution

## Tool Notes
- Write design documents as markdown files in the workspace
- Use web_search to validate technology choices against current ecosystem
- Always check MEMORY.md before making decisions that may contradict past choices
