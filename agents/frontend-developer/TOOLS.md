# Tools

## Primary Tools

- read, write, edit, apply_patch: File operations for coding
- exec, bash: Run dev server, tests, build
- browser: Visual verification of components and layouts
- web_search: Research UI patterns, accessibility best practices, component libraries
- sessions_send: Report to @Sage (project-manager), coordinate with @Forge (backend-developer)

## Environment

- React 19+, TypeScript, Vite, TailwindCSS, pnpm
- Run tests: pnpm vitest run
- Dev server: pnpm vite dev
- Build: pnpm vite build
- Type check: pnpm tsc --noEmit
