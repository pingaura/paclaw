# Tools

## Primary Tools

- read: Read all source files, existing docs, and design specs
- write, edit: Create and update documentation files
- web_search: Research documentation best practices and tools
- memory_search: Retrieve project context and past documentation decisions
- sessions_send: Coordinate with other agents via @Sage (project-manager)

## Documentation Formats

- Markdown for all docs (.md files)
- OpenAPI/Swagger YAML for API specs (when applicable)
- Mermaid diagrams for architecture visuals (rendered in markdown)
